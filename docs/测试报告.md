## 3. 发现的问题

### 3.1 严重问题 (需要立即修复)

1. **EnhancedConfigManager 导入失败**
   - 问题: `name 'ABC' is not defined`
   - 影响: 系统配置无法加载
   - 建议: 检查模块中的抽象基类导入

2. **MainController 导入失败**
   - 问题: `No module named 'bayes_opt'`
   - 影响: 主控制器无法启动
   - 建议: 将 bayes_opt 设为可选依赖或添加 try-except 处理

### 3.2 中等问题 (建议修复)

3. **策略评估器方法参数不匹配**
   - 问题: 多个计算方法参数数量不匹配
   - 影响: 策略评估功能受限
   - 建议: 统一方法签名

4. **策略组合优化器数据类型错误**
   - 问题: `'float' object has no attribute 'get'`
   - 影响: 无法执行组合优化
   - 建议: 检查数据结构定义

5. **事件系统数据库表未创建**
   - 问题: `sqlite3.OperationalError: no such table: events`
   - 影响: 事件持久化失败
   - 建议: 确保数据库初始化时创建表

### 3.3 轻微问题 (可选修复)

6. **前端构建警告**
   - 问题: chunk 文件大于 500KB
   - 影响: 加载性能
   - 建议: 使用代码分割优化
