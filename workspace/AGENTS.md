# AGENTS.md - 交易系统工作空间

## 安全警告
⚠️ 这是加密货币交易系统工作空间，包含敏感的交易密钥和策略。
⚠️ 不要在这个目录中存储无关文件，避免信息泄露。

## 工作流程
### 1. 日常启动
1. 检查系统状态和网络连接
2. 验证交易所API密钥状态
3. 加载最新的市场数据
4. 检查持仓和订单状态

### 2. 交易执行流程
1. 接收交易策略和参数
2. 验证市场条件和风险水平
3. 计算最优交易价格和数量
4. 执行交易并确认
5. 记录交易日志和更新持仓

### 3. 风险控制
1. 每日亏损限额：2%总资产
2. 单笔交易风险：≤1%资产
3. 最大持仓比例：≤30%资产
4. 强制止损：-5%入场价

## 文件组织
- `scripts/` - 交易脚本和自动化工具
- `references/` - API文档和市场分析
- `crypto-config/` - 加密的配置文件（API密钥等）
- `memory/` - 交易日志和绩效记录
- `strategies/` - 交易策略配置文件

## 外部依赖
1. **交易所API**：通过环境变量设置API密钥
2. **市场数据**：实时行情源（WebSocket/API）
3. **数据库**：交易记录和绩效数据存储
4. **监控系统**：系统健康检查和预警

## 紧急处理
1. **系统异常**：立即停止所有交易活动
2. **API失效**：切换到备用API或暂停交易
3. **市场异常**：触发紧急止损和平仓
4. **网络中断**：记录状态等待恢复

## 备份策略
1. 每日备份交易记录到加密存储
2. 每周备份策略配置和历史数据
3. 每月完整系统快照和性能分析
