## 🆘 **遇到问题？**

### **常见问题解决：**

1. **Python版本问题**
   ```bash
   # 检查Python版本
   python3 --version
   
   # 如果版本低于3.11，安装3.11
   sudo apt install -y python3.11
   
   # 创建虚拟环境时指定版本
   python3.11 -m venv venv
   ```

2. **Docker权限问题**
   ```bash
   # 添加用户到docker组
   sudo usermod -aG docker $USER
   
   # 需要重新登录生效
   # 或使用sudo运行docker命令
   sudo docker-compose up -d
   ```

3. **数据库连接问题**
   ```bash
   # 检查PostgreSQL服务
   docker-compose ps postgres
   
   # 检查端口占用
   netstat -tulpn | grep :5432
   
   # 重启服务
   docker-compose restart postgres
   ```
